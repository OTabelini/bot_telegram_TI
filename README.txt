ABOUT THE BOT

It's a telegram bot, made for provide career information (focused in data analytics). 
With this bot, the HR can access the bot to know you better, download your cv and other basic informations.
In the bot_telegram file you'll find the code documented to guide you to change whatever you want to make this tool worth it for you.

PIP INSTALL 
To make the bot run, you have to pip install pytelegrambotapi on your terminal
